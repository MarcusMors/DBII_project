## Credenciales de la base de datos
crea un archivo `.env` y pon tus credenciales y el uri que vas a usar
por ejemplo:
```txt
NEO4J_URI=bolt://localhost:7687
NEO4J_USERNAME=your_username
NEO4J_PASSWORD=your_password
```
